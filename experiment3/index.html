Experiment 3 - Generative Methods

Imitate

It took me a while to think of something I wanted to create for generative arts. By the end, I had something in mind that I wanted to create. My goal in mind is to merge the morphing shape by sepatolo, and with the wandering point and trail effect from the Random Walk that Wes Modes created from the slides.

The work of Sepatolo consists of generative that continuously moves and changes shape. You can move around with your mouse cursor. The image can be shown below, and the script is available at this link. click here

The random walk by Wes Modes consists of perlin noise that selects a random direction and moves a set distance in that direction. The script is available at this link. click here

Overall, the first script creates a morphing shape using points on a circle, while script 2 involves a point wandering around the canvas, and I want to combine them together to see how it would turn out.

Integrate

Most of my script will be based on integrating the two scripts I have chosen from step 1. First, I want to create a trailing effect by introducing the two arrays (xOld, yOld) from the random walk script to store the history of positions for each point. From this, I initialize these arrays to store the positions of the points.

>

In the draw function, I want to adjust the logic to update the position of each point with random moves while maintaining the trailing history and drawing the trails by connecting previous positions.

From the image above, I added an ellipse to draw the points. It’s found on the top left of my screen. To make the points more accessible to other places, I want to add a user interaction, where I would implement a mousePressed() function where I can reset the shape’s position with a mouse click. Which can both script and image found below

I updated my script with random x and y, similar to in random walk, to create a swigly or random drift. In addition, I want to draw my trailing lines by looping the line from the last point to the current point. I also added a stroke from a lighter color and a map function to start the trail brighter and the end of the trail fainter, reading a fading effect as the points move. But for that to happen, I included unshift, which creates a trail effect by keeping track of where the points have moved. You can see the results below.

The output looked and felt small and slow for me, so I want to make some adjustments, such as changing the background for a trailing effect and framerate to make it faster. Also, I deleted the e function from setapolo and added it directly to my setup function to make my script look cleaner. It is hard to see on a screenshot, so I will leave it as it is for now.

Innovate

It is tough for me to think about what I can do to innovate. I think I just want to make this art some sort of aesthetic reasoning. I added a pop method on the image below because I want to remove the tail once it exceeds the specified length. I want to ensure that the trail only shows the most recent movements. The result shows the trail after leaving in a minute. Compared to the image above, you can see that the oldest position got popped.

I also changed the background color from 255 to 0 at my setup function to leave any trails when I moved my circle. I also changed the color of my line by adding HSB color mode and color variation. Lastly, I added a dynamic radius based on my mouse position. As you can see in the last picture, the line radius is much bigger if your mouse is away from your keyboard. Reflection

I worked on this project by myself. Working on this project is an exciting experience. There are a lot of concepts to think of. I am very proud to use such methods as pop and unshift to make a trail. I am slowly getting used to learning how to use JavaScript. Having no constraints or limitations helps me create what I want. There’s no such thing as a grading script that limits what I want to make this happen. My highs are during my innovative part. It is pretty straightforward from what I did to make this work. I only have to add a pop function to leave no trail behind, adding color hues and dynamic radius for mouse movement. My low is integrating two scripts during integration. I had to search on greeks4geeks, chatGPT, and other resources on how to create a trail, and I found out the stroke, and the unshift to make it happen. Overall, I am satisfied with the work I’ve done.
