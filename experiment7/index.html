Experiment 7: Data Visualization & Networks

Imitate

We have discussed various creative image ideas for this assignment. With that in mind, we thought about creating a scatter plot with data as our starting point. We also considered using the top 100 anime on MyAnimeList as our main dataset for plotting.

Integrate

We decided to create a force-directed graph to visualize the anime data. We designed each node, representing an anime, to repel each other to avoid overlap. Additionally, we added an interactive hover feature that displays the details of each node. We created a node class to encapsulate the information of each anime. Also, we added color intensity based on the popularity of the anime. The image below shows us interacting with a node, with details visible.

Innovate

In this part, we decided to create a visual effect where nodes are linked by lines when they are close to each other. This works by checking the distance between the nodes and drawing a line when they are within range.
